Tina is not apache

Welcome to Tina, a simple web server for linux computers. The current release is Tina-0.1.0.

Tina is a single-tasking HTTP server. That means that unlike traditional web servers, it does not fork for each incoming connection, nor does it fork many copies of itself to handle multiple connections. It internally multiplexes all of the ongoing HTTP connections, and forks only for CGI programs (which must be separate processes.)

Tina is not intended as a feature-packed server,it's implementation is simple.Tina uses the non-blocked select mode for clients' connection,and pthread for each process.The I/O process uses evbuffer from libevent,and fork for CGI program.
